# Contributing

## Convenciones

- Canon de contenido: `SCRIPTORIUM_ARCONTICO/`.
- Nombre de archivo: `snake_case.md`.
- Un tema por archivo.
- Estado editorial en frontmatter: `borrador|revision|canon`.

## Flujo de PR

1. Crea rama desde `main`.
2. Edita/crea usando plantillas en `_templates/`.
3. Verifica Markdown localmente (opcional) y abre PR.
4. Completa checklist de PR.
5. Espera CI verde y revision.
