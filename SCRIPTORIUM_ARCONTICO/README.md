# SCRIPTORIUM_ARCONTICO

Indice canonico del Scriptorium.
Este directorio es la fuente oficial para contenido editorial.

## Mapa de rutas

- `00_MANIFIESTO/`: fundamentos, axiomas y marco de identidad.
- `01_GLOSARIOS/`: definiciones operativas y semanticas.
- `03_LIBRO_REFLEJOS_HIBRIDOS/`: estructura y materiales del libro.
- `05_PROMPT_RITUAL/`: canon de prompts y operadores rituales.
- `_templates/`: plantillas de produccion.

## Regla de canon

- Todo contenido nuevo nace aqui.
- `arconte/` se mantiene como capa de compatibilidad temporal.
- En caso de conflicto, prevalece `SCRIPTORIUM_ARCONTICO/`.
