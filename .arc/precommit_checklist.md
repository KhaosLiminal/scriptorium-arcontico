# Pre-Commit Checklist Arc v1.0

## Validacion Ontologica

- [ ] Existe TKN explicito.
- [ ] Los 7 sellos estan completos.
- [ ] El modo arcontico esta declarado.

## Validacion Editorial

- [ ] Esta alineado con Canon.
- [ ] No duplica prompt existente sin motivo.
- [ ] Impacto declarado en changelog.

## Validacion Tecnica

- [ ] Paso por POT documentado.
- [ ] Parser/flujo validado.
- [ ] No redefine semantica TKN.

## Validacion Productiva

- [ ] Tiene trazabilidad hasta TKN.
- [ ] Tiene metrica/KPI definido.
- [ ] No compromete arquitectura base.
